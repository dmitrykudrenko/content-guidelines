## Як працювати з цими файлами

- **Розширення** — кожен файл можна допрацьовувати окремо, не чіпаючи інші
- **Адаптація** — файли зараз universal; для кожного продукту можна додати окремий файл або секцію
- **Статус** — використовуй позначки `TODO`, `DRAFT`, `DONE` у кожному файлі для трекінгу прогресу
